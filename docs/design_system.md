## 3. Сетка и отступы

- Артборды: `428 x 926` dp.
- Боковые поля контента: `(428 - 396) / 2 = 16` dp.
- Карточки рецепта расположены вертикальной колонкой; внешний отступ снизу-сверху между карточками — определяется автолейаутом frame `102:3` (требует уточнения через `itemSpacing` родителя; в текущей реализации используем 8 dp сверху и снизу = 16 dp зазор).
- Высота status bar: `35` dp.
- Высота bottom navbar: `60` dp.
- Базовая единица сетки: 4 dp.

## 4. Карточка рецепта (`Карточка рецепта`, component `116:33`)

Точные размеры из макета:

- Габариты: `396 x 136` dp.
- Фон: `#FFFFFF`.
- Радиус скругления: `5` dp (одинаковый по всем углам у фона; у фото — только левые: `[5, 0, 0, 5]`).
- Тень: `DROP_SHADOW` blur `4`, offset `(0, 4)`, цвет `rgba(149,146,146,0.10)`.
- Фото рецепта: `149 x 136` dp, **слева, на всю высоту карточки**, скругление только левых углов `5 dp`. `BoxFit.cover`.
- Текстовый блок справа от фото:
  - Название: Roboto 500, 22/22, `#000000`, ширина текстового бокса 208 dp, до 2 строк, обрезка ellipsis.
  - Длительность: Roboto 400, 16/18.75, `#2ECC71`. Слева от текста — иконка часов 16x16 dp (растровая, в макете `IMAGE`).
- Внутренние отступы текстового блока относительно правого края фото — определяются позиционированием, базовая величина ~`16` dp слева от фото и сверху.

## 6. Bottom navbar (`navbar login`, component_set `121:443`)

Габариты: `428 x 60` dp. Фон: `#FFFFFF`. Тень: `DROP_SHADOW` blur `8`, offset `(0, 0)`, цвет `rgba(0,0,0,0.25)` — мягкий ореол вокруг бара. Подписи: Roboto 400/10/23, иконки 24x24 dp. Активный таб (текст + иконка): `#2ECC71`; неактивный: `#C2C2C2`.

В component_set 6 вариантов, объединённые в две раскладки:

- **logIn (4 таба)**: «Рецепты» (иконка `пицца`), «Холодильник» (иконка `холодильник`), «Избранное» (иконка `червы`/сердце), «Профиль» (иконка `пользователь`). Один из 4 вариантов соответствует активной странице — у соответствующей подписи цвет `#2ECC71`, у остальных `#C2C2C2`.
- **logOut (2 таба)**: «Рецепты» (иконка `пицца`) и «Вход» (иконка `пользователь`). Использован на splash, login и страницах рецептов logOut-ветки.

Иконки в макете — растровые ассеты `icons8-*-24`. Для имплементации заменяем на 4 SVG в `assets/icons/nav/`:
`pizza.svg`, `fridge.svg`, `heart.svg`, `user.svg`.

## 9d. Бейдж «Закладка» (instance `145:226`)

- Габариты: `66 x 23` dp.
- Контур: `BOOLEAN_OPERATION subtract` (вырезан треугольный «хвост»). Эффективная заливка `#2ECC71` после вычитания `Polygon 1` 34×27.
- Тень: `DROP_SHADOW` blur `6`, offset `(0, 4)`, `rgba(0,0,0,0.25)`.
- Число: Roboto w800 14/16.4, `#FFFFFF`.
- В коде проще всего реализовать через `CustomPainter` или `Stack` с `ClipPath` (стрелка вниз).

## 12. Что в макете отсутствует / требует решения

- Auto-layout `itemSpacing` между карточками не выгружен в текущем срезе — в коде используем 8 dp сверху и снизу.
- Hover/pressed-цвета не заданы (по умолчанию ripple Material).
- Тёмная тема не описана.
- Состояния «загрузка», «ошибка», «пусто» в макете не нарисованы — берём из практики (`CircularProgressIndicator(color: primary)`).
- «Закладка» в макете — boolean-subtract из двух фигур (прямоугольник минус треугольник). В Flutter воспроизводим через `ClipPath` или `CustomPainter`.
- Анимации и длительности переходов в Figma не заданы — splash 2 с + 600 мс fade выбран в коде.
- Иконки `icons8-*` — растровые; в проде обязательно заменяем на SVG.
- Размер форм-контейнера «Начать готовить» выравнивание по центру экрана — выведено из координат инстанса.
